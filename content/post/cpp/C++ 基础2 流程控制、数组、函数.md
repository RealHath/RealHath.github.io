# 3 跳转语句
## 3.1 break
作用：跳出整个循环

跳出for2，执行for1循环
```cpp
for1()
{
    for2()
    {
        break;
    }
}
```

## 3.2 continue
作用：结束一次循环

跳出for2的一次循环，继续执行for2循环
```cpp
for1()
{
    for2()
    {
        continue;
    }
}
```

## 3.3 goto
跳转到标记，往下执行

不加限制地使用 GOTO 语句，特别是使用往回跳的 GOTO 语句，会使程序结构难于理解，在这种情形，应尽量避免使用 GOTO 语句。
```cpp
语句1
goto flag;
语句2

flag:
语句3
```
